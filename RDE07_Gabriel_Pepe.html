Branches

São em suma, ramificações de um projeto de software ou documento (sendo uma forma mais segura se editar esse arquivo) e tem como objetivo o desenvolvimento paralelo de:

  • Novas de funcionalidades;
  • Correção de problemas;
  • Alteração de código.

    • O projeto pode ser colocado no:

    Branch Master ou Main: Braço principal de um projeto em um repositório.

    Branch Secundária: Braço secundário de um projeto em um repositório.

    Hospedagem

    Hospedamos nosso site através do GitHub pages, selecionando um branch.
    Utilizaremos esse sistema pois ele oferece hospedagem gratuita de sites.
    Basta criar uma conta e um repositório público e a URL será: https://nomeusuario.github.io/repositorio
